quota¶

A python package for solving thrust allocation problems in the plane. The implementation is plenty fast enough to be used in real-time applications.

Utilizes quadratic programming with linearized constraints and disjunctive methodology together with (optional) slack variables for fast and robust allocation.

Module containing the core allocation solver functionality

quota.constraints.concatenate_constraints(original_set, additional_set)¶
Method for concatenating sets of linear constraints.

original_set and additional_set are both tuples of for (C, b, n_eq). Output is a concatenated tuple of same form.

All equality constraints are always kept on top.

class quota.thruster.AzimuthThruster(pos, max_force, n_discret)¶

Bases: quota.thruster.Thruster

Convenience class for setting up a typical

Azimuthing thruster.

It is assumed that an Azimuthing thruster can

produce the same amount of force in any direction [0,2*pi]. The only bound existing is then the maximum force that can be delivered (max_force).
